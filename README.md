# 🥪 週末熱壓吐司經營模擬器 (Weekend Toast Tycoon) - V3

這是一個專為「小資副業營運」設計的即時模擬計算器。透過深藍色調的專業介面，協助使用者精算週末販售熱壓吐司的獲利能力，並加入了殘酷的「房價增值對比」功能，讓你一眼看透勞動收入與資產增值的距離。

![Version](https://img.shields.io/badge/version-3.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 🌟 核心功能

### 1. 即時營運儀表板
* **多維度獲利分析**：即時顯示「當日」與「每月」的營業額、成本及純利。
* **小資版成本結構**：預設採用優化後的食材配置（1/8 鮪魚、1/3 洋蔥、平價起司與吐司），單份成本精確控制在 **$32.9**。

### 2. 動態經營拉桿
* **營業時間設定**：可調整每日 0-8 小時的經營時長。
* **客流量與售價控制**：模擬每小時 0-20 人的客流量，並彈性設定單份售價 ($50-$70)。

### 3. 😂 殘酷對比區 (V3 新增)
* **房價複利計算**：以 3000 萬房產為基底，計算年增 3% 的複利增值空間。
* **通膨連動模擬**：模擬隨年分增加，食材成本與售價每年 2% 的通膨增長。
* **勞動比例對比**：自動計算「吐司利潤」佔「房價增值」的百分比，反映真實的資產差距。

## 🚀 快速開始

1. 下載 `V1.html` 與 `para.json`。
2. 直接使用瀏覽器開啟 `V1.html` 即可運行（無需後端環境）。
3. 調整拉桿，開始規劃你的週末副業。

## 🛠 技術棧

* **Frontend**: HTML5, CSS3 (CSS Variables, Flexbox, Grid)
* **Logic**: Vanilla JavaScript (ES6+)
* **Data**: JSON 格式食材清單

## ⚖️ 免責聲明

本工具之計算結果僅供參考。實際營運成本可能因地區物價波動、設備損耗及洋蔥切碎程度而有所不同。房價增值亦受市場景氣影響，不保證每年固定 3% 成長。
